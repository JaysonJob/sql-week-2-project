## What this covers

- **Part 1 – String Functions:** Using UPPER, LOWER, LENGTH, SUBSTR, and CONCAT to manipulate and format text data from the `nairobi_academy` schema.
- **Part 2 – Number Functions:** Using ROUND, CEIL, FLOOR, COUNT, AVG, MAX, MIN, and SUM to perform calculations on exam results.
- **Part 3 – Date & Time Functions:** Extracting date parts, calculating age, and formatting dates using PostgreSQL date functions.
- **Part 4 – SQL JOINs:** Using INNER JOIN, LEFT JOIN, RIGHT JOIN, and multi-table JOINs to query related data from the `city_hospital` schema.
- **Part 5 – Window Functions:** Using ROW_NUMBER(), RANK(), DENSE_RANK(), NTILE(), AVG() OVER, and LAG() to perform advanced analytics on exam results.
- **Part 6 – SET Operators:** Using UNION, UNION ALL, and INTERSECT to combine results across the `nairobi_academy` and `city_hospital` schemas.
